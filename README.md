# オーバーウォッチ2 勝敗カウンター

## これは何？

- OBSのWebSocketから画像を取得し、事前に学習したデータから勝利/敗北を自動推論
- 推論した結果をカウントし、OBSの配信・録画画面へ出力
- (CSSの知識があれば)出力デザインはある程度カスタマイズ可能

## 制約事項

- FHD(1920×1080) サイズのみ対応
- 言語は英語もしくは日本語のみ対応
- ライバルマッチ(俗にいうランクマッチ)のみ対応
